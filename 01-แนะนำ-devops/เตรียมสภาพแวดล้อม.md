## **การติดตั้ง ubuntu 24.04 lts desktop**

โหลดได้ที่ Microsoft Store&#x20;

<div align="left"><figure><img src="../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure></div>

```bash
sudo apt update && sudo apt -y upgrade
sudo apt -y install ca-certificates curl gnupg git
# Docker
curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER
# Compose v2 มาพร้อม docker รุ่นใหม่
# Node.js LTS
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt -y install nodejs
```

> ออกจากระบบแล้วเข้ากลับเพื่อให้สิทธิ์ docker กลุ่มทำงาน
