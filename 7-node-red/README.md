# บทที่ 7 Node RED

> **Node-RED** เป็นแพลตฟอร์มแบบ Low-code ที่ช่วยให้การพัฒนาระบบอัตโนมัติและการจัดการข้อมูลเป็นเรื่องง่าย ผ่านการเชื่อมต่อโหนดต่าง ๆ เช่น Input, Processing และ Output เหมาะสำหรับการทำงานร่วมกับอุปกรณ์ IoT, ฐานข้อมูล และระบบ Backend เพื่อสร้าง Workflow ที่ยืดหยุ่นและขยายต่อได้ง่าย

### วัตถุประสงค์ของบทนี้

* เข้าใจแนวคิดการทำงานของ Node-RED
* สามารถออกแบบ Workflow แบบอัตโนมัติ
* เชื่อมต่อ Node-RED กับอุปกรณ์ IoT และ Backend
