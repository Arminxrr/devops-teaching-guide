# บทที่ 5 - ติดตั้ง GitLab + Runner

> **GitLab** และ **GitLab Runner** เป็นองค์ประกอบสำคัญของระบบ DevOps ที่ช่วยให้กระบวนการทำงานแบบ CI/CD เป็นอัตโนมัติ ตั้งแต่การจัดการซอร์สโค้ด การตรวจสอบคุณภาพ ไปจนถึงการ Deploy ระบบ โดย GitLab ทำหน้าที่เป็นศูนย์กลางในการเก็บโค้ดและบริหารจัดการ Pipeline ส่วน GitLab Runner ทำหน้าที่รันงานตามที่กำหนดไว้ในไฟล์ `.gitlab-ci.yml` เพื่อให้การทำงานเป็นไปอย่างมีประสิทธิภาพและเป็นระบบ

#### **วัตถุประสงค์ของบทนี้**

* เข้าใจบทบาทของ GitLab และ GitLab Runner ในระบบ DevOps
* สามารถติดตั้ง GitLab Server บนระบบ Ubuntu ได้อย่างถูกต้อง
* สามารถติดตั้งและตั้งค่า GitLab Runner ให้พร้อมใช้งาน
* เชื่อมต่อ GitLab Runner กับ GitLab Server
* เตรียมระบบสำหรับการใช้งาน CI/CD ในบทถัดไป
