# บทที่ 2 - ติดตั้ง GitLab + Runner

> &#x20; การติดตั้ง **GitLab** และ **GitLab Runner** เป็นขั้นตอนสำคัญของระบบ DevOps เพื่อทำให้การทำงานแบบ CI/CD มีความอัตโนมัติ ตั้งแต่การจัดการโค้ด การตรวจสอบคุณภาพ ไปจนถึงการ Deploy ระบบ โดย GitLab ใช้เป็นศูนย์กลางเก็บโค้ดและจัดการ Pipeline ส่วน GitLab Runner ทำหน้าที่รันงานที่ถูกกำหนดไว้ใน `.gitlab-ci.yml`

หลังติดตั้ง เข้าเว็บ `http://gitlab.local` → Login ด้วยผู้ใช้ `root`&#x20;
