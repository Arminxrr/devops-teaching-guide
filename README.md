# หน้าแรก

> คู่มือภาษาไทยสำหรับการสอนและฝึกปฏิบัติ DevOps แบบครบวงจร\
> ครอบคลุมตั้งแต่การติดตั้งเครื่องมือ พัฒนาแอป (Node.js) ใช้ Docker/Compose, ทำ GitLab CI/CD, Deploy จริง, ไปจนถึง Monitoring

## หัวข้อหลัก

1. แนะนำ DevOps และการเตรียมเครื่อง
2. ติดตั้ง GitLab CE และเชื่อม GitLab Runner
3. พื้นฐาน Docker + เขียน Backend ตัวอย่าง
4. เขียน Pipeline CI/CD บน GitLab
5. Deploy ระบบจริง + Nginx Reverse Proxy
6. Monitoring ด้วย Prometheus + Grafana + Loki
7. สรุปภาพรวมและไฟล์ Template
