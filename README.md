# หน้าแรก

> คู่มือภาษาไทยสำหรับผู้ที่อยากเข้าใจ **DevOps ตั้งแต่พื้นฐานจนถึงการ Deploy ระบบจริง**\
> ครอบคลุมการใช้งาน Docker, GitLab CI/CD, Backend, Frontend\
> พร้อมตัวอย่างที่สามารถนำไปใช้งานได้จริง

### สิ่งที่จะได้จากคู่มือนี้

* เข้าใจแนวคิด **DevOps ตั้งแต่พื้นฐาน**
* สร้าง Server และ Virtualization ด้วย **Proxmox**
* ใช้งาน **Docker & Docker Compose**
* ติดตั้งและใช้งาน **GitLab + GitLab Runner**
* เขียน **CI/CD Pipeline** ด้วย GitLab
* พัฒนา **Backend & Frontend**
* Deploy ระบบจริง&#x20;
* ได้ตัวอย่างไฟล์และ Template สำหรับใช้งานจริง

***

### โครงสร้างเนื้อหาโดยสรุป

1. แนะนำ DevOps และการเตรียมเครื่องมือ
2. ติดตั้ง Server และ Proxmox
3. ติดตั้ง Ubuntu บน Virtual Machine
4. Docker และ Docker Compose
5. ติดตั้ง GitLab และ GitLab Runner
6. การทำงานของ GitLab CI/CD
7. Node-RED และระบบ Automation
8. ระบบ Backend
9. ระบบ Frontend
10. Deploy ระบบจริง
11. สรุปภาพรวมและไฟล์ Template

***

### แนวทางการใช้งานเอกสาร

* แนะนำให้เรียนและทำตาม **เรียงตามลำดับบท**
* ทุกบทออกแบบให้สามารถ **ทำตามได้จริง**
* เหมาะสำหรับใช้เป็น **รายงาน / โปรเจกต์ / คู่มือฝึกงาน**
