# บทที่ 8 -Backend

> **Backend** คือส่วนสำคัญของระบบแอปพลิเคชันที่ทำหน้าที่ประมวลผลข้อมูล จัดการตรรกะการทำงาน และเชื่อมต่อกับฐานข้อมูลหรือบริการภายนอก บทนี้จะอธิบายแนวคิดการพัฒนา Backend การออกแบบ API และการเตรียมระบบ Backend ให้พร้อมสำหรับการนำไปใช้งานร่วมกับ Docker, CI/CD และระบบ Production

#### **วัตถุประสงค์ของบทนี้**

* โครงสร้างการทำงานของ Backend&#x20;
* สามารถออกแบบและใช้งาน API ได้อย่างเหมาะสม
* เตรียม Backend ให้พร้อมสำหรับการใช้งานร่วมกับ Docker และ CI/CD
* เชื่อมต่อ Backend กับระบบอื่น เช่น Frontend, IoT&#x20;

***

**การนำ Backend ไปใช้งาน**

* เชื่อมต่อกับ Frontend&#x20;
* รองรับการทำงานของระบบ IoT และ Automation
* ทำงานร่วมกับ Docker และ Docker Compose
* Deploy ผ่าน GitLab CI/CD ไปยัง Production
