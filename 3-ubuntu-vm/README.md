# บทที่ 3 -ติดตั้ง Ubuntu บน VM

> **Ubuntu** เป็นระบบปฏิบัติการแบบ Linux ที่ได้รับความนิยมสูงในงาน Server และ DevOps เนื่องจากมีความเสถียร ปลอดภัย และรองรับเครื่องมือจำนวนมาก บทนี้จะอธิบายขั้นตอนการติดตั้ง Ubuntu บน Virtual Machine (VM) เพื่อใช้เป็นพื้นฐานสำหรับการติดตั้งระบบต่าง ๆ ในบทถัดไป เช่น Docker, GitLab และ CI/CD

#### **วัตถุประสงค์ของบทนี้**

* เข้าใจแนวคิดของการใช้งาน Virtual Machine
* สามารถติดตั้ง Ubuntu บน VM ได้อย่างถูกต้อง
* ตั้งค่าระบบพื้นฐานหลังการติดตั้ง Ubuntu
* เตรียมสภาพแวดล้อมสำหรับการใช้งานด้าน Server และ DevOps
* พร้อมต่อยอดไปยังการติดตั้งระบบในบทถัดไป
